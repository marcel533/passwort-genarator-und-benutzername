# Passwort- & Benutzername-Generator

Du brauchst schnell ein sicheres Passwort oder einen neuen Benutzernamen?
Dieses Tool erledigt das für dich.

Mit einem Klick bekommst du starke, zufällige Passwörter, die nicht leicht zu knacken sind.
Wenn dir kein guter Name einfällt, hilft dir der Generator mit kreativen Benutzernamen, die nicht nach Standard aussehen.

Keine Anmeldung.
Keine Daten werden gespeichert.
Einfach nutzen und weitergehen.

Ideal für alle, die keine Zeit oder Lust haben, sich jedes Mal neue Passwörter oder Namen auszudenken.
